Python Rules

  • Indent 4 spaces always when using statements
  • A Function is 'def'
  • The import function allows you to import other modules which are hidden functions it runs
  • The input function allows the users to type in some input
  • A variable can be Represented as just the name_of_variable = str,int or bool
  • The print statement will show up in the shell as feedback from the program
